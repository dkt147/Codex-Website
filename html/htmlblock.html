Every HTML element has a default display value, depending on what type of element it is. There are two display values: block and inline.

Block-level Elements

A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can).
The <div> element is a block-level element.

<div>Welcome to Codex</div>

Here are some block level elements:
<address>    <article>    <aside>    <blockquote>    <canvas>    <dd>    <div>
   <dl>    <dd>    <dt>    <figcaption>    <figure>    <footer>    <form>    <h1>-<h6>
<header>    <li>    <main>    <hr>    <nav>    <noscript>    <ol>    <p>    <pre>    <section>    <table>    <tl>    <tfoot>    <video>

Inline Elements

An inline element does not start on a new line and it only takes up as much width as necessary. This is a <span> element inside a paragraph.

<span>Welcome to Codex </span>

Here are some Inline elements:
 <a>    <abbr>    <acronym>    <b>    <bdo>    <big>    <br>    <button>    <side>
   <code>    <dfn>    <em>    <i>    <img>    <input>    <kbd>    <label>    <map>
   <object>    <output>    <q>    <samp>    <script>    <select>    <select>    <span>    <strong>    <sup>    <sub>    <textarea>    <time>    <tt>    <var>
